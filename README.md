## 🚀 Como Utilizar

- Entre na pasta raiz do projeto: `cd`
- Execute o comando: `./gradlew bootrun`

<em>** Visando facilitar a demostração da aplicação, recomendo instalar apenas o IntelliJ IDEA e executar o projeto através da IDE **</em>
